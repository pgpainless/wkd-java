# Command Line Interface for the Web Key Directory

```shell
Usage: wkdcli fetch [-a] USERID
Fetch an OpenPGP Certificate via the Web Key Directory
      USERID    Certificate User-ID
  -a, --armor   ASCII Armor the output
```
